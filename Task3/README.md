# User Registration + Display Cards

This project is a simple **React JS application** where users can fill out a registration form and the submitted data is displayed as **user cards** on the screen.

## Features

* User registration form
* Add user data to a list
* Display registered users as cards
* Dynamic UI updates using React state

## Tech Stack

* React JS
* Vite
* Tailwind CSS

## Live Demo

https://we-lab-task-3.vercel.app/

## Run Locally

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Build for production:

```
npm run build
```
