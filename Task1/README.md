# Task 1: Counter with History

## Overview
This project is Task 1 of the Final Comprehensive Practice Projects. It focuses on mastering React state management by building a counter that tracks its own history.

## Tech Stack
*Framework*: React

*Build Tool*: Vite

*Styling*: Tailwind CSS

## Features
Increment / Decrement: Modify the counter value by 1 in real-time.

Soft Reset: Returns the counter to 0 while preserving the history log.

Hard Reset: Wipes both the current value and the entire history list.

History Tracking: A dynamic list that logs every previous state change.
