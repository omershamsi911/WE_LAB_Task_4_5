# University Portal

This project is a **React JS University Portal application** integrating multiple components with routing and shared state.

## Features

* Navigation between pages
* Student list displayed in a table
* Add new student form
* Student profile cards
* State lifting for shared data
* Page navigation using React Router

## Tech Stack

* React JS
* React Router
* Vite
* Tailwind CSS
