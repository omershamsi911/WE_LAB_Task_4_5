# Quiz Application

This project is a **React JS quiz application** with score tracking and navigation between questions.

## Features

* 5 quiz questions
* Next and previous navigation
* Score tracking
* Routing between quiz pages

## Tech Stack

* React JS
* React Router
* Vite
* Tailwind CSS

## Live Demo

https://we-lab-task-7.vercel.app/

## Run Locally

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Build for production:

```
npm run build
```
