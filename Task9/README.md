# Shopping Cart

This project is a **React JS shopping cart application** where users can add items, remove items, update quantities, and see the total price.

## Features

* Add items to cart
* Remove items from cart
* Update item quantity
* Automatic total price calculation

## Tech Stack

* React JS
* Vite
* Tailwind CSS

## Live Demo

https://we-lab-task-9.vercel.app/

## Run Locally

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Build for production:

```
npm run build
```
