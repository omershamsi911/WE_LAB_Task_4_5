# Blog Mini-App

This project is a **React JS blog mini application** that displays blog posts as cards and allows users to view the full post on a separate detail page.

## Features

* Display blog posts as cards
* View single post details
* Routing between pages
* Static blog content

## Tech Stack

* React JS
* React Router
* Vite
* Tailwind CSS
