# Dashboard with Tabs

This project is a **React JS dashboard interface** with multiple tabs controlled using React state.

## Features

* Dashboard layout
* Tabs for Profile, Settings, and Orders
* State-controlled active tab switching
* Dynamic content rendering

## Tech Stack

* React JS
* Vite
* Tailwind CSS

## Live Demo

https://we-lab-task-10.vercel.app/

## Run Locally

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Build for production:

```
npm run build
```
