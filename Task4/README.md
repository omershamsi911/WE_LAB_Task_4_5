# Product Catalog with Search

This project is a simple **React JS application** that displays a **product catalog** and allows users to **search and filter products by name or price**.

## Features

* Static product catalog
* Search products by name
* Filter products by price
* Dynamic filtering using React state

## Tech Stack

* React JS
* Vite
* Tailwind CSS
* JavaScript (Hooks)

## Live Demo

https://we-lab-task-4.vercel.app/

## Run Locally

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Build for production:

```
npm run build
```
